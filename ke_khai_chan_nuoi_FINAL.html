📖 Hướng dẫn kết nối Google Sheets
1

Tạo Google Spreadsheet

Vào sheets.google.com → Tạo bảng tính mới → Đặt tên Kê Khai Chăn Nuôi

2

Mở Apps Script

Trong Sheets → Tiện ích mở rộng → Apps Script → Xóa code cũ → Dán code bên dưới

3

Triển khai Web App

Triển khai → Quản lý các bản triển khai → ✏️ (Sửa) → Phiên bản: Phiên bản mới
Chọn: Ai có quyền truy cập: Bất kỳ ai → Triển khai → Cấp quyền.
⚠️ Sửa bản triển khai cũ để giữ nguyên URL (không tạo bản mới).

4

Chia sẻ link/QR cho người dân

Lưu file HTML này lên Google Sites hoặc nhờ đơn vị IT tạo hosting miễn phí trên GitHub Pages, Netlify... rồi lấy link chia sẻ Zalo, Facebook.

🏘️ Cập nhật danh sách thôn
📱 Mã QR kê khai (cho người dân)

In mã QR này dán lên bảng thông báo thôn hoặc đăng Facebook để người dân quét và kê khai. (QR mở chính trang kê khai này.)
