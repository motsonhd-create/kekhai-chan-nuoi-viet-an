📖 Hướng dẫn kết nối Google Sheets
1

Tạo Google Spreadsheet

Vào sheets.google.com → Tạo bảng tính mới → Đặt tên Kê Khai Chăn Nuôi

2

Mở Apps Script

Trong Sheets → Tiện ích mở rộng → Apps Script → Xóa code cũ → Dán code bên dưới

3

Triển khai Web App

Triển khai → Triển khai mới → Ứng dụng web
Chọn: Ai có quyền truy cập: Mọi người → Triển khai → Cấp quyền → Copy URL dán vào ô trên

4

Chia sẻ link/QR cho người dân

Lưu file HTML này lên Google Sites hoặc nhờ đơn vị IT tạo hosting miễn phí trên GitHub Pages, Netlify... rồi lấy link chia sẻ Zalo, Facebook.
